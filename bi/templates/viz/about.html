Hi, I'm My!

Welcome to my portfolio! This is a Django project that manages all of my dashboards on the backend, while each dashboard is a Plotly Dash app.

Why Plotly Dash?

It's an open source framework with plenty of built-in functions and graphs for creating interactive dashboards. Especially, Plotly provides the ability to freely customize graphs and charts based on your needs, unlike most BI platforms which you probably find some limitations. Plus, No experience in HTML, CSS and Javascript is needed to build a web application.

Why Django?

Again, it's an open source framework! It wasn't too challenging for me to develop this web app thanks to simple syntax, ORM (Object Relational Mapper), multiple built-in packages and database backends. I'm also able to manage multiple dashboards at once, for example, showing or hidding specific dashboards to viewers without having to change a single line in my code.
